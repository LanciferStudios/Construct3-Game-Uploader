A simple plugin to upload and display Construct 3 HTML5 games on WordPress.

== Description ==
Upload your Construct 3 games to WordPress with ease. Features include game ZIP uploads, rich text descriptions, orientation options (Landscape, Portrait, or Custom sizes), shortcodes, auto-generated pages, a fullscreen button, and basic game management.

== Installation ==
1. Download the ZIP from this repository.
2. In WordPress, go to `Plugins > Add New > Upload Plugin`, and upload the ZIP.
3. Activate the plugin.
4. Navigate to “Construct3 Games” in the admin menu to add your games.

== Requirements ==
- PHP 7.4 or higher with ZIP extension enabled.
- A Construct 3 game ZIP with an `index.html` file.
